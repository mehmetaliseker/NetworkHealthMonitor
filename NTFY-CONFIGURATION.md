# ntfy Configuration

ntfy ayarlari UI icindeki `Ayarlar > ntfy` ekranindan yapilir.

Gerekli alanlar:

- Sunucu URL'i
- Topic
- Gerekliyse token veya authorization bilgisi

Token loglanmamalidir ve release ZIP icine alinmamalidir. Gercek kabul testinde test bildiriminin gercek subscriber tarafinda alindigi dogrulanmalidir.
